# Markdown Note Editor

A minimal markdown note editor built with React, TypeScript, and Vite.

## Overview

This app is designed for fast note-taking with a clean, minimal interface.

High-level features:
- Live markdown editing and preview
- Resizable editor and preview panels
- Auto-save while you work
- Multiple view modes: edit, split, and preview
- Searchable note sidebar
- Wiki-style note links like `[[note_name]]`
- Light and dark theme support

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router
- CodeMirror 6

## Quick Start

Install dependencies:

```bash
npm i
```

Run in development:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Notes

This project currently includes `dev`, `build`, and `preview` scripts. There is not a separate `start` script at the moment.
